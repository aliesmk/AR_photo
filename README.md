# AR Photo Frame

<p dir="rtl">
<strong>پروژه واقعیت افزوده برای پخش ویدیو روی عکس‌های چاپی</strong><br>
با استفاده از OpenCV، تشخیص ویژگی (Feature Matching) و Homography، ویدیو را روی عکس مرجع در تصویر دوربین زنده قرار می‌دهد.
</p>

---

## قابلیت‌ها

- **موتور AR دسکتاپ (MVP):** تشخیص عکس مرجع از طریق دوربین وب‌کم و پخش ویدیو روی آن
- **سرور مدیریت سفارش:** پنل ادمین با احراز هویت (ورود با نام کاربری/رمز) برای آپلود عکس و ویدیو و مدیریت سفارشات
- **PWA موبایل قابل نصب:** اپلیکیشن تحت وب با طراحی مدرن (Dark + Neon Purple) برای تشخیص AR
- **احراز هویت دو مرحله‌ای:** ورود با شماره موبایل + کد تأیید ۴ رقمی (کد پیش‌فرض: ۱۲۳۴)
- **محدودیت بازدید:** قابلیت تنظیم محدودیت تعداد بازدید برای هر سفارش (مناسب مدل درآمدی)
- **ویدیوی خصوصی:** قابلیت تنظیم ویدیو به صورت خصوصی با دسترسی فقط برای افراد مجاز
- **مدیریت دسترسی:** مالک می‌تواند کاربران مجاز را اضافه/حذف کند
- **ثبت بازدید:** لاگ کامل بازدیدها با قابلیت مشاهده آمار
- **QR Code امن:** هر سفارش دارای QR Code با امضای HMAC برای امنیت
- **فشرده‌سازی ویدیو:** فشرده‌سازی خودکار ویدیوها با FFmpeg برای پخش بهینه
- **پشتیبانی از دیتابیس:** ذخیره‌سازی سفارشات، کاربران و لاگ‌ها در PostgreSQL
- **پشتیبانی از HTTPS:** اجرای امن سرور با گواهی SSL

## نصب و راه‌اندازی

### پیش‌نیازها

- Python 3.10+
- PostgreSQL 14+
- FFmpeg (برای فشرده‌سازی ویدیو)
- دوربین (وب‌کم یا دوربین موبایل)

### 1. نصب وابستگی‌های پایتون

```bash
pip install -r requirements.txt
```

### 2. تنظیم محیط (Environment)

```bash
cp .env.example .env
```

فایل `.env` را با مقادیر صحیح پر کنید.

### 3. تنظیم دیتابیس PostgreSQL

```sql
CREATE DATABASE ARP;
```

دیتابیس به صورت خودکار توسط SQLAlchemy ساخته می‌شود (در صورت نیاز، جداول قبلی حذف و بازسازی می‌شوند).

پس از اجرای سرور، مرورگر موبایل را به آدرس سرور باز کنید.

## مدل درآمدی

این پروژه از **محدودیت بازدید** برای مدیریت هزینه‌های سرور استفاده می‌کند:

1. هر سفارش می‌تواند **محدودیت تعداد بازدید** داشته باشد (مثلاً ۲۰۰ بار)
2. پس از اتمام محدودیت، ویدیو به صورت خودکار غیرفعال می‌شود
3. کاربر می‌تواند با **تمدید** (خرید مجدد) محدودیت را افزایش دهد
4. ویدیوهای بدون تمدید پس از **حدود ۱ ماه** از سرور حذف می‌شوند
5. محدودیت پیش‌فرض: **۱۰۰ تا ۳۰۰ بازدید رایگان**

### ویدیوی خصوصی

- ویدیوهای خصوصی فقط برای **مالک** و **افراد مجاز** قابل مشاهده است
- مالک می‌تواند از طریق اپلیکیشن، کاربران جدید اضافه یا حذف کند
- افراد مجاز با شماره موبایل شناسایی می‌شوند
- ویدیوهای عمومی برای همه قابل مشاهده است

## امنیت QR Code

هر QR Code حاوی یک امضای HMAC-SHA256 است:
- فرمت: `order_uuid:signature`
- امضا با کلید خصوصی سرور ساخته می‌شود
- از جعل QR Code جلوگیری می‌کند
- نیازی به جستجوی دیتابیس در زمان اسکن ندارد
